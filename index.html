Рефлюкс Контроль (YTRF)

Образовательный YouTube-канал о гастроэзофагеальной рефлюксной болезни (ГЭРБ) — для пациентов, их семей и медицинских специалистов.

Проект ООО «Технодело» — российский имплантируемый аппарат для хирургического лечения ГЭРБ. Цель: #1 русскоязычный ресурс о лечении рефлюкса.

▶ Образовательные разборы, интервью, Q&A
▶ Язык: RU (EN субтитры)
▶ 8-20 мин (образование), 30-60 мин (интервью)
▶ Средний темп, медицинская анимация

Этот гайдлайн определяет визуальные экранные типы для стадии Pre-Edit. Каждый тип имеет назначение, ограничения по тексту и ключевые слова. Шахматный паттерн = зона спикера.
